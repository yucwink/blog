1.快捷键

1.1代码编辑

  • ctrl+shift+/ 块注释

  • ctrl + d 删除行

  • ctrl + shift + u 切换大小写

  • ctrl + shift + v 打开剪切板，从历史记录粘贴

  • ctrl + enter 开始新行

  • ctrl + alt + L 格式化代码

  • alt + shift + 上下箭头 上下移动行

  • alt+shift+C 最近的更改

  • ctrl + shift + backspace 跳转到最后编辑的位置

1.2代码阅读

  • 全局查找 ctrl + shift + F
  • 搜索一切 shift x 2
  • 运行一切 ctrl x 2
  • 切换标签 alt + 箭头
  • 切换标签 ctrl + tab
  • 最近文件 ctrl + E
  • 查看形参 ctrl + P
  • 前往声明 ctrl + B
  • 前往实现 ctrl + alt + B
  • 方法切换 alt + 上下箭头
  • 跳转行列 ctrl + G

1.3 文件操作

  • 复制路径 ctrl + shift + C
